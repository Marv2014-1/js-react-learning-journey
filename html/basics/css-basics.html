CSS Basics

Understanding how to style HTML

1. What is CSS?

CSS (Cascading Style Sheets) describes how HTML elements are displayed. It controls layout, colors, fonts, spacing, and more.

2. Ways to Apply CSS

  • Inline: <h1 style="color: red;">
  • Internal: inside <style> tags
  • External: via linked .css file (recommended)

4. Common Properties

  • color: text color
  • background-color: background color
  • font-size: size of text
  • margin, padding: spacing outside/inside
  • border: outline of elements
