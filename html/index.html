HTML Basics

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content using tags and elements.

1. Basic Document Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    Page content goes here.
  </body>
</html>

2. Headings

Headings go from <h1> (largest) to <h6> (smallest):

<h1>Main Title</h1>
<h2>Subheading</h2>

3. Paragraphs and Text Formatting

<p>This is a paragraph.</p>
<strong>Bold text</strong>
<em>Italic text</em>

4. Links and Images

<a href="https://example.com">Visit Site</a>
<img src="image.jpg" alt="Description">

5. Lists

Unordered List:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Ordered List:

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

6. Tables

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>30</td>
  </tr>
</table>

7. Forms

<form action="/submit" method="POST">
  <label>Name:</label>
  <input type="text" name="name">
  <input type="submit" value="Send">
</form>

8. Semantic Elements

These give meaning to your layout:

<header></header>
<nav></nav>
<main></main>
<section></section>
<article></article>
<footer></footer>

9. Embedding Media

<video controls>
  <source src="movie.mp4" type="video/mp4">
</video>

<audio controls>
  <source src="sound.mp3" type="audio/mp3">
</audio>

10. Comments

<!-- This is a comment -->

Note: HTML is usually combined with CSS (for styling) and JavaScript (for interactivity) to build complete websites.
