Block vs Inline Elements

Every HTML element has a default display value.

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

An inline element does not start on a new line. An inline element only takes up as much width as necessary.

Div

For this reason (and since div is a block element and takes all avilable width), div is usally used as a container to hold a group of elements

Class

The class attribute is used to point to a class name in a stylesheet or by javascript to manipulate variables.

Multiple elements can share the same class
